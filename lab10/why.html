Why is Indexed Database API Important?

Indexed database API is so important it has its own W3C Recommendation!

Indexed DB may link multiple object stores (lists) and multiple databases, making it easier to search and more efficient to find results that are unique to your query! This is because of the many unique attributes a value can hold.

Indexed DB API is also important because it is persistent, meaning the API will hold the information on your side of the website, even when you are away!
