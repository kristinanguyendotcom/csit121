How is Indexed Databse API Used?

Indexed Database API has a B Tree data formation, and each database has a set of object stores.

Object stores are lists holding more data, similiar to SQL in key/value pairs. The persistent feature of Indexed DB is great for retaining data in the browser even when you are away. For exmaple, when you're shopping online with a cart or when you're logged in and it remembers your preferences and digital self.

It serves an important role in storage of information on the client/user end and makes for a smoother and positive user experience.
